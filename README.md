# Robot-scara

Interfaz gráfica para el control del robot SCARA.

## Características

Este proyecto proporciona una GUI básica en Python con Tkinter que permite:

- Dibujar trayectorias en un canvas interactivo
- Capturar coordenadas (x, y) en tiempo real
- Limpiar el canvas para reiniciar el dibujo
- Guardar las coordenadas en archivos de texto

## Inicio Rápido

```bash
# Ejecutar la aplicación GUI
python3 scara_gui.py

# Ejecutar la demostración
python3 demo_gui.py

# Ejecutar las pruebas
python3 test_scara_gui.py
```
